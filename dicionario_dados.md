# Dicionário de Dados

## FT_VENDAS

| Campo | Descrição |
|---------|-----------|
| segment | Segmento de mercado |
| country | País |
| product | Produto |
| sales | Receita líquida |
| profit | Lucro |
| units_sold | Quantidade vendida |
| discounts | Descontos |
| cogs | Custo dos produtos vendidos |

## FT_BUDGET

| Campo | Descrição |
|---------|-----------|
| budget_sales | Receita orçada |
| budget_profit | Lucro orçado |

| Campo | Descrição |
